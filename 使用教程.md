# 3x-ui Cloudflare 部署器使用教程（零基础可用）

这篇教程教你如何在一台已安装 `3x-ui` 的 VPS 上，快速完成：

- 按需创建 `VLESS / Trojan / VMess` 节点
- 通过 **SQLite 直写（旧版默认）** 或 **REST API（新版可选）** 管理入站
- 自动配置 Cloudflare `DNS + SSL + Origin Rules`
- 自动生成可用的优选订阅链接
- 支持一键卸载并回滚配置

---

## 一、前置条件

开始前请先确认以下条件：

1. VPS 上已安装并可正常运行 `3x-ui`
2. 如果没有安装，请先执行下面命令安装：

```bash
bash <(curl -Ls https://raw.githubusercontent.com/mhsanaei/3x-ui/master/install.sh)
```

3. **旧版默认**：存在 `/etc/x-ui/x-ui.db`，脚本直写数据库
4. **新版可选**：面板 API 可用，并具备用户名密码或 API Token
5. 你有 Cloudflare 账号邮箱和 Global API Key

## 二、在服务器上执行安装命令

直接复制以下命令（不依赖 git）：

```bash
command -v python3 >/dev/null 2>&1 || (sudo apt update && sudo apt install -y python3)
curl -fsSL -o xui_cf_deployer.py https://raw.githubusercontent.com/byJoey/xui-cf-deployer/main/xui_cf_deployer.py
chmod +x xui_cf_deployer.py
sudo python3 xui_cf_deployer.py
```

## 三、安装模式交互说明

脚本启动后会先让你选择模式：

- `1`：安装（默认）
- `2`：卸载

安装时依次输入：

1. 绑定域名（例如：`node.example.com`）
2. Cloudflare 邮箱
3. Cloudflare Global API Key（隐藏输入）
4. 创建协议 (1=vless,2=trojan,3=vmess，逗号分隔，留空 = 全部)：`1,2`

脚本会自动检测 x-ui 版本，选择数据库直写或 API，无需手动选模式。

协议编号：

- `1 = VLESS`
- `2 = Trojan`
- `3 = VMess`

## 四、部署成功后会发生什么

脚本会自动完成以下动作：

1. 生成 UUID、随机端口、随机路径
2. 写入 3x-ui（数据库直写 或 API，取决于上一步选择）
3. 重启 x-ui / Xray
4. Cloudflare 创建/更新 A 记录并开启代理（橙云）
5. Cloudflare SSL 设置为 `flexible`
6. 下发 Origin Rules（按路径转发到对应端口）
7. 输出所选协议的订阅链接

## 五、订阅链接参数说明（博客可选展示）

生成链接会带以下基础参数：

- `epd=yes`
- `epi=yes`
- `egi=no`
- `dkby=yes`

协议开关策略为：

- 当前协议：`yes`
- 其他协议：`no`

例如输出 VLESS 时会是：

- `ev=yes`
- `et=no`
- `evm=no`

并附带 URL Encode 后的 `path` 参数。

## 六、如何卸载并回滚

如果你要清理本次配置，重新运行脚本并选择卸载模式：

```bash
sudo python3 xui_cf_deployer.py
```

模式输入：

- `2`（卸载）

卸载会自动执行：

- 删除上次创建的 3x-ui 入站
- 恢复 Cloudflare Origin Rules
- 恢复 Cloudflare SSL 配置
- 恢复或删除对应 DNS 记录
- 删除本地状态记录
